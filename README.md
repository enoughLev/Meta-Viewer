# Meta-Viewer
Приложение для извлечения метаданных из изображений.
## Возможности приложения
- Выбор файла для получения метаданных
- Просмотр выбранного изображения
- Работа с ранее сохраненными изображениями
- Отображение геопозиции, извлеченной из изображения
## Сборка программы 
Для работы понадобятся Python версии 3.10, а также добавление необходимых зависимостей:

    python install -r requirements.txt
Далее можно запускать программу:

    python main.py

### Запуск скомпилированного файла
Также можно воспользоваться уже скомпилированным файлом. В нем программа имеет идентичный функционал с исходным проектом и способна работать самостоятельно. Файл находится в папке dist/. Название "MetaViewer.exe".
    
Для самостоятельно компиляции проекта можно воспользоваться командой:

    pyistaller main.spec
Команда подгружает конфигурацию для сборки из файла main.spec и компилирует код в exe-файл.
